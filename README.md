## Fundamentos de Ciência de Dados - Aplicação de Técnicas de IA

### Objetivo
Utilizando os dados disponíveis em http://archive.ics.uci.edu/ml/, dados
Iris, foi aplicado as técnicas pedidas abaixo para classificação de padrões:
- Vizinho mais próximo;
- K vizinhos mais próximos (k=3 e k=5);
- DMC;
- k-means;
  
Para cada algoritmo:
- mínimo, máximo, média e desvio padrão para os percentuais de acertos (algoritmos executados 20 vezes e variados de 10% a 80%);
